INTRODUCCION- IMPORTANCIA DE GIT Y GITHUB


Git es importante porque nos permite tener el control de todos los cambios en nuestro codigo

Github es importante ya que nos permite trabajar de forma remota y ordenada sin necesidad de emplear otras herranientas.

de igual forma permite publicar un proyecto y compartirlo con la comunidad para esperar la colaboracion de otras personas o desarrolladores.


HISTORIA DE GIT Y GITHUB

GIT fue creado por Linus TOrvalds en 2005 para el desarrollo del kerner linux, junto a otros desarrolladores

contribuyó al desarrolo inicial. Actualmennte el proyecto es mantenido por Junio Hamano desde 2005.

GITHUB fue desarrolado por Chris Wanstrath, p.j. Hyett, tom preston-werner y Scott Chacon usando ruby on Rails, y empezó en 2008. Aunque la compañia GitHub, Inc, existia desde 2007

el 24 de febrero anunciaron que en su primer añor habian logrado acumulas 46.000 repositorios publicos, 17.000 de los cuales fueron creados en solo 1 mes.


COMO INSTALAR GIT

-Ve a https://git-scm.com/ y descarga el paquete de instalacion

-debes elegir la opción según la arquitectura de tu computadora (32bit o 64bit)

-Ahora debes ejecutar el archivo descargado y verás una pantalla como esta

-elige la carpeta donde ubicaras los archivos de git

-Asegúrate de tener seleccionado git bash que es la herramienta principal con la que trabajaremos y con esto se terminará la instalación

-Ahora tendras disponible Git Bash deste tu lista de programas, aquí es donde trabajaremos con Git, ábrelo.

-si escribes git --version y presionas enter y no ves un error aparente y la consola muestra un resultado similar a este todo está perfecto y podrás seguir todo el curso sin el menor problema.
